# NUKG Mini Project - URL Shortener

## Overview

This project is a simple URL Shortener web application developed using Python and Flask. It allows users to convert long URLs into short and easy-to-share links. When a user accesses a shortened URL, they are automatically redirected to the original website.

## Features

* Shortens long URLs into compact links
* Generates unique short codes
* Redirects users to the original URL
* Simple and user-friendly interface
* Built using Flask framework
* Lightweight and easy to deploy

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Jinja2 Templates

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Akshayanaidi/NUKG-mini-project.git
```

### 2. Navigate to the Project Directory

```bash
cd NUKG-mini-project
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```text
http://127.0.0.1:5000
```

## How It Works

1. User enters a long URL.
2. The application generates a unique short code.
3. The short URL is displayed to the user.
4. When the short URL is accessed, the application redirects to the original URL.
